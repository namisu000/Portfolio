Object Model Viewer

#OpenGL #GLSL #OBJ

OpenGL로 OBJ 3D 모델을 읽어 렌더링하는 뷰어. 대규모 메시의 법선 계산 병목을 자료구조로 개선한 컴퓨터그래픽스 과제입니다.

역할

문제 구현하며 마주친 문제

다양한 크기의 OBJ 모델을 화면 중앙에 일정한 크기로 배치해야 했습니다.
정점 법선 계산이 모든 면을 순회하는 방식이라, 정점·면이 많은 모델에선 실행이 멈출 만큼 느려졌습니다.

개선 해결하고자 한 노력

정점 min/max로 중심을 구해 원점으로 이동하고, 대각선 길이로 스케일을 정규화해 다양한 모델을 동일 화면 크기로 표시했습니다.
면 법선(cross product)·정점 법선을 계산하고, Phong 셰이딩(vertex/surface normal 토글, specular·shininess 조절)을 적용했습니다.
정점 법선 계산을 vertexToFaceMap(unordered_map)으로 바꿔, 정점에 연결된 면만 탐색하도록 개선해 전체 순회를 제거했습니다.
배운 점 — vertexToFaceMap은 효과는 확실했지만 단순 매핑에 의존한 방식이라 더 정교한 자료구조를 쓰지 못한 점이 아쉬웠습니다. 자료구조·알고리즘 설계가 렌더링 성능에 직접 영향을 준다는 걸 체감했습니다.

Spherical Panorama Viewer

#OpenGL #GLSL #환경매핑

360° 파노라마 이미지를 구체에 매핑해 둘러보는 뷰어. 환경 매핑·Fresnel·Diffuse Light Map을 셰이더로 구현한 컴퓨터그래픽스 과제입니다.

역할

문제 구현하며 마주친 문제

반사 벡터를 구면 좌표로 변환해 360° 텍스처를 정확히 매핑해야 했습니다.
카메라 pitch가 90°를 넘으면 화면이 좌우 반전되는 문제가 있었습니다.

개선 해결하고자 한 노력

관찰 벡터를 법선으로 반사시켜 반사 벡터를 구면 좌표(atan/asin)로 변환하고, 수평은 fract로 반복·수직은 clamp로 제한해 환경 매핑을 구현했습니다.
구면 좌표계(radius·yaw·pitch) 카메라를 구성하고, pitch에 따라 up 벡터를 갱신해 90° 반전 문제를 해결했습니다.
관찰각 기반 Fresnel 근사로 환경 반사를 mix하고, 법선 방향 텍스처 좌표로 Diffuse Light Map을 샘플링했습니다.
배운 점 — Diffuse Light Map 초기 구현에서 텍스처 좌표를 반사 방향으로 계산해 원하는 결과가 안 나왔고, 법선 벡터로 바꿔 해결했습니다. 수식 하나의 차이가 결과를 바꾼다는 걸 직접 확인했습니다.

Gaussian Splat Viewer

#OpenGL #GLSL #3DGS #ComputeShader

외부 3DGS 엔진 없이 OpenGL 셰이더 파이프라인으로 구현한 가우시안 스플랫 뷰어. 깊이 정렬 병목을 GPU로 옮겨 회전 시 끊김을 해소했습니다.

기간
2025.05 – 2025.06
기술
C++ · OpenGL · GLSL · GLM
역할
개인 프로젝트
상태

문제

깊이 정렬을 CPU에서 매 프레임 동기적으로 수행해, 시점을 회전할 때 화면이 뚝뚝 끊겼습니다.

개선

정렬 전 과정(prefix-sum·scatter)을 compute shader 기반 GPU radix sort로 구현해 GPU에 상주시켰습니다. 끊김이 사라지고 정렬 처리량이 크게 개선돼 프레임 저하가 해소됐습니다.

작업 범위 — 정식 3DGS의 EWA 투영·타일 래스터라이저는 범위 밖으로, 지오메트리 셰이더 빌보드 근사와 하드웨어 블렌딩으로 대체했습니다.

3D 영상 기반 모션 추출

#SMPL-X #모션복원

단안 영상에서 SMPL-X 전신 모션을 복원하는 파이프라인. 손목 회전의 관측 불가능성(null space)을 swing-twist 직교 분해로 풀어낸 창의경진대회 우수상 프로젝트입니다.

기간
2026.04 – 2026.06
기술
Python · SMPL-X
역할
팀 · GVHMR·HaMeR fusion
성과
창의경진대회 우수상

문제 구현하며 마주친 문제

HaMeR는 가려진 손에 약하고, GVHMR은 손목 axial 회전을 추론하지 못했습니다.
손목 axial 회전은 단안 2D에서 관측 불가능한 null space라 단순 결합으로는 풀리지 않았습니다.

개선 해결하고자 한 노력

손목 회전을 swing(관측 가능, HaMeR)·twist(관측 불가, forearm)로 직교 분해해 간섭 없이 처리했습니다.
GVHMR(body)·HaMeR(hand)을 5단계 graft로 결합해 SMPL-X 시계열을 생성했습니다.
temporal 안정성을 보존해 jitter를 7.8°에서 3.6°로 줄였습니다.
배운 점 — 모델을 검증 없이 채택한 실수에서 출발해, 이후 모델 도입 전 논문·실행 검증을 절차로 고정했습니다. 관측 불가능성을 수학적으로 분해하는 접근을 배웠습니다.
